# Machine Learning Journey 🚀

Welcome to my Machine Learning Journey repository! This project tracks my daily progress, conceptual deep dives, and mathematical implementations of foundational machine learning algorithms from scratch and using scikit-learn.

## 📅 Day-by-Day Progress

### 🔹 Day 1: Linear Regression Fundamentals
* **Concepts:** Manual simple linear regression math, core cost function mechanics, and basic model fitting.
* **Files:**
  * `Linear Regression.ipynb`
  * `Simple linear regression by formula.ipynb`

### 🔹 Day 2: Gradient Descent Fundamentals & Full From-Scratch Coding
* **Concepts:** Deep dive into gradients, mathematical optimization, partial derivatives, and troubleshooting common implementation bugs.
* **Files:**
  * `Gradient Descent fundamentals.ipynb`
  * `Gradient descent (2).ipynb`
  * `grad_desc_fully_coded.ipynb`

### 🔹 Day 3: Batch & Mini-Batch Gradient Descent
* **Concepts:** Scaling optimization techniques, understanding computational efficiency trade-offs, and batching datasets for vectorization.
* **Files:**
  * `batch-gradient-descent.ipynb`
  * `mini-batch-gradient-descent-from-scratch.ipynb`

### 🔹 Day 4: Stochastic Gradient Descent & Multiple Linear Regression
* **Concepts:** Updating parameters per single data point, dealing with noisy convergence, and expanding feature matrices for multiple input variables.
* **Files:**
  * `Multiple-LR.ipynb`
  * `stochastic-gradient-descent-from-scratch.ipynb`

### 🔹 Day 5: Regularization & Coding Challenge
* **Concepts:** Preventing overfitting using L1 (Lasso) and L2 (Ridge) penalties, balancing both via ElasticNet, and consolidating skills through an algorithmic coding challenge.
* **Files:**
  * `Ridge.ipynb`
  * `Ridge_lasso_elasticnet.ipynb`
  * `coding-challenge.ipynb`
